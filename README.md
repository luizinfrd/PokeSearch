# Pokedex

Uma Pokedex simples feita com HTML, CSS e JavaScript, utilizando a API pública [PokeAPI](https://pokeapi.co/).

## Como usar

1. Abra o arquivo `index.html` em seu navegador.
2. Use o campo de busca para procurar Pokémon pelo nome.

## Estrutura do Projeto
- `index.html`: Estrutura principal da aplicação.
- `style.css`: Estilos visuais.
- `script.js`: Lógica de integração com a PokeAPI.

## API
- [PokeAPI](https://pokeapi.co/)
